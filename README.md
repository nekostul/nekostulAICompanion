# AI Companion NPC

Мод для Minecraft с NPC-напарником, который помогает игроку:
добывает ресурсы по команде, переносит предметы, работает с вёдрами и сундуком,
следует за игроком и защищает его от враждебных мобов.

Мод **полностью работает без ИИ**.  
Интеграция с ИИ планируется как дополнительная возможность.

---

## Возможности

- NPC следует за игроком и защищает его
- Выполняет команды игрока через чат
- Добывает запрошенные ресурсы (дерево, земля, камень и др.)
- Использует вёдра для воды и лавы (сначала просит вёдра)
- Сбрасывает добытые предметы рядом с игроком
- Использует персональный сундук при переполнении инвентаря
- Работает оффлайн, без API-ключей

---

## Примеры команд

- `добудь дерево`
- `принеси землю`
- `принеси 2 ведра воды`
- `следуй`
- `стоп`
- `помощь`
- `помощь сундук`

NPC выполняет **только то, что попросил игрок**.
